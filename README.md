# Women Safety Project

A **web application to enhance women's safety**, providing safety tips, SOS alerts, incident reporting, and awareness resources. Built using **Flask, HTML, CSS, and JavaScript**.  

---

## **Features**

- **Women Safety Tips** – Informative tips for staying safe.  
- **SOS Emergency Button** – Trigger an alert with location to emergency contacts.  
- **Incident Reporting** – Users can report incidents with description, severity, and optional evidence.  
- **User Authentication** – Registration and login system with secure password hashing.  
- **Safety Quiz** – Quick awareness quiz to educate users.  
- **Dashboard** – Admin dashboard to view reports, SOS alerts, and visualize report statistics with charts.
